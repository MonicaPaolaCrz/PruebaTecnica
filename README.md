# Vue.js Web Application - User Management with Camera and Dark/Light Mode

## Descripción del Proyecto

Este proyecto es una aplicación web desarrollada con **Vue.js** que consume una API pública para mostrar una lista de usuarios.

## Funcionalidades

- **Consumo de API**: Utiliza **Axios** para obtener 50 registros de usuarios de la API pública.
- **Filtrado en tiempo real**: Permite filtrar la lista de usuarios por nombre sin necesidad de recargar la página.
- **Eliminación de columnas**: Puedes eliminar usuarios de la tabla con una transición visual en tiempo real.
- **Cámara**: Un botón para activar la cámara del dispositivo y tomar fotos, que se guardan temporalmente.
- **Modo claro/oscuro**: Cambia entre el modo claro y oscuro de la interfaz.

## Tecnologías Usadas

- **Vue.js** (Framework principal)
- **Axios** (Biblioteca para realizar solicitudes HTTP)
- **CSS** y **Bootstrap 5** (Para el diseño y estilo)
## Requisitos

- **Node.js** v16 o superior
- **NPM** (Node Package Manager)
